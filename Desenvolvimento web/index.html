Desenvolvimento Web

1

Curriculo HTML

Um dos primeiros projetos que realizei foi a criação do meu currículo, utilizando HTML e CSS. Este projeto me permitiu aplicar e aprimorar meus conhecimentos em desenvolvimento web, aprofundando meu entendimento sobre HTML e CSS.

2

Calculadora

Outro dos meus projetos que realizei foi o desenvolvimento de uma calculadora usando HTML, CSS e JavaScript. Este projeto foi uma excelente oportunidade para aplicar meus conhecimentos em desenvolvimento web, melhorar minhas habilidades nessas tecnologias e adquirir conhecimento sobre pilha e fila. Além disso, pude aprofundar meu entendimento sobre a interação entre HTML, CSS e JavaScript.

3

Atividade com iFrame

Desenvolvi um projeto que utiliza iframes para diversas telas interativas. Inclui uma calculadora padrão e uma calculadora de INSS, bem como tabelas para dados pessoais como nome, CPF e senha. Também implementei uma seção de scripts que, combinando HTML, CSS e JavaScript, exibe resultados dinâmicos ao lado. Esta experiência me permitiu explorar diversas funcionalidades e aprofundar meu conhecimento em desenvolvimento web.

4

Pokédex

Desenvolvi uma Pokédex com HTML, CSS e JavaScript, integrando a PokéAPI para exibir números, nomes e GIFs animados de Pokémon. Esse projeto me permitiu aprofundar meus conhecimentos em HTML, CSS para estilização e JavaScript para funcionalidades. Estou entusiasmado para compartilhar essa experiência e continuar explorando mais sobre desenvolvimento web.
