Automação industrial

1

Semáforo

O projeto coordena os semáforos para veículos e pedestres, garantindo segurança e fluidez no trânsito. Quando o sinal está verde ou amarelo para veículos, é vermelho para pedestres, e vice-versa. Antes de mudar para vermelho, o sinal dos pedestres pisca três vezes como aviso.

2

Relógio

Este projeto é um relógio digital exibido em um LCD, que conta e mostra segundos, minutos e horas. Ele proporciona uma maneira clara e precisa de acompanhar o tempo em formato digital.

3

Prova

Este projeto é um programa feito com LCD, com uma tela inicial de boas-vindas e um menu com três opções: um contador que registra quantas vezes o botão azul é pressionado; um relógio digital onde o botão azul aumenta a velocidade e o amarelo a restaura; e uma opção para zerar todos os dados.
